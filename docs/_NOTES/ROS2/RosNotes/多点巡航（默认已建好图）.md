## 多点巡航（默认已建好图）

1. **记录航点**

   1.1 启动底盘（ros2 launch yahboomcar_nav laser_bringup_launch.py）

   1.2 虚拟机上启动rviz2（ros2 launch yahboomcar_nav display_nav_launch.py）

   1.3 启动导航节点（ros2 launch yahboomcar_nav navigation_dwa_launch.py）

   1.4 录制节点（python3 src/yahboomcar_nav/yahboomcar_nav/record_goal_pose.py -o src/yahboomcar_nav/yahboomcar_nav/bags/my_waypoints.yaml）

   1.5 初始化后就开始发布位置（2D Goal Pose），record_goal_pose.py会自动记录航点，结束后保存为bags/my_waypoints.yaml文件

2. **启动巡航**

   2.1 启动底盘（ros2 launch yahboomcar_nav laser_bringup_launch.py）

   2.2 虚拟机上启动rviz2（ros2 launch yahboomcar_nav display_nav_launch.py）初始化位置

   2.3 启动导航节点（ros2 launch yahboomcar_nav navigation_dwa_launch.py）

   2.4 发布航点（ros2 run yahboomcar_nav waypoint_navigator --file /home/jetson/yahboomcar_ros2_ws/yahboomcar_ws/src/yahboomcar_nav/bags/my_waypoints.yaml）
